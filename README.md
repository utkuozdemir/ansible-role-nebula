Ansible Role: nebula
=========

Ansible role to install [Nebula](https://github.com/slackhq/nebula) Mesh.

Requirements
------------

The requirements are:
- Ansible version >=2.10
- Linux
- Systemd as init system

This role is tested on:
- Ubuntu 20.04 Focal Fossa
- Python 3.10
- Ansible 2.10
